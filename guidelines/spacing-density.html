Density — the size axis

Set data-cs-density="compact" on any container to tighten control metrics (button + field heights and paddings) for dense product surfaces — tables, admin consoles, toolbars. Layout spacing tokens are untouched; type is untouched. Comfortable is the default and needs no attribute. On coarse pointers (touch) compact is intentionally inert — the 44px target floor from base/a11y.css stays authoritative.

Comfortable default

button md 44px · sm 36px · field 44px

Compact data-cs-density="compact"

button md 36px · sm 30px · field 38px (fine pointer)

Rules — density composes with every other axis (Theme × Element × Expression × Language); state it on the nearest container, never encode it in component code. Tokens live in tokens/density.css; the density gate asserts compact applies and defaults stay untouched.
