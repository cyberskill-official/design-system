support.js runtime identity gate

Every template folder ships a support.js copy of the Claude Design DC runtime (the compiler's contract: one byte-identical copy per template). This gate fetches all of them (driven by _ds_manifest.json — currently 84), sha-256s each with SubtleCrypto, and asserts exactly one unique hash and a copy count equal to the template count. A second hash means a template's runtime was hand-edited or a stale copy survived a regeneration. Verdict on window.__supportid.
