# CyberSkill Live (Storybook)

This is the **single Live hub** for operators: Theme × Element × Language toolbar, full component CSF with control matrices, and **Live/Surfaces** for Motion, Identity Lab, templates, kitchen-sink, AI cluster, and RTL.

## Navigation

- **Components/** — every public React primary with Default + matrix stories
- **Live/Surfaces** — portable HTML iframes (Atomic View, Motion, Identity Lab, …)
- Toolbar — same axes as product templates

## Portable consumers

Product apps still use `styles.css` + bundle/ESM — never this Storybook. Zero-build Atomic View remains at `guidelines/atomic-view.html` for gates and clone-and-open.

See `docs/storybook.md` and `docs/live-view-vs-storybook.md`.
